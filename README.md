# Zenith Lab Portfolio

A premium one-page portfolio concept for a multi-disciplinary developer, featuring:

- Executive hero messaging
- Case-study styled work grid
- Technical proficiency stack section
- "Beyond the Code" off-duty gaming segment
- Scroll progress bar and subtle parallax motion

## Run locally

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.
